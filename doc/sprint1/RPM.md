During the meeting the team went over all the user stories and selected all the ones we will finish for the release. The stories that are bold are the ones we selected to implement in this release. The meeting was short, and so another meeting will be held tomorrow for sprint1 and the story points.

**As a user who has never been to the library I would like to create a Library account/card so that I can start borrowing books.**

**As John (a regular at the library) I would like to log in to the library website to check out books under my library account.** 

**As John I would like to be able to search for books by any of the books attributes (Author/Title/Genre) so that I can find a book that matches my interests.** 

**As John I would like to see if a book is available, and if it's not, when it is expected to become available so that I can plan when I can read the books I’m interested in.**

**As John I would like to check out an available book so I can read it.** 

**As John I would like to reserve an unavailable book so that I can read it as soon as it becomes available.**

**As John I would like to be able to extend a due date for a book I’ve checked out so that I have more time to read it in case I am not going to be done by the time it’s due.**

**As John I would like to see all the books I’ve checked out and their due dates so that I can return them on time.**

**As Myriam (the librarian) I would like to add or remove books from the collection displayed online to make damaged or missing books unavailable or make newly added books available.**

**As Myriam I would like to be able to mark a book as checked in online when someone brings it back in person, so that someone else can check it out later.**

As Myriam I want to be able to check which account has a particular book checked out at the moment so that I can get in contact with them in case they have long outstanding books or return a book damaged.

**As a user who didn’t find the specific book they wanted I would like to request a book to be added to the library so that I can eventually read it without buying it.**

**As Myriam I would like public users to be automatically charged and prevented from checking out more books if they have outstanding books that are past due to minimize the chances of losing more books, and to not have to do it manually.**

As John I would like to publish reviews for books I’ve read (checked out) so that I can share my opinion on the book with others.

As John I would like to see other peoples reviews of books so that I can decide whether I want to read a specific book or not.

**As a forgetful patron of the library I would like to be sent reminders when a due date for a book I have checked out is approaching so that I remember to bring it back to the library on time.**

As John I would like to have access to ebooks that I can access online and so that I don’t have to commute to the library to pick up a book.

As John I would like for my ebooks to become automatically unavailable once they’re past due so that I don’t have to worry about manually checking them in.

**As a forgetful patron of the library I would like to be able to check if I owe a fee to the library so that I know how much money I should take with me when I return the book.**

**As John I would like to access a list of all the books I’ve previously checked out so that I can keep track of all the books I’ve read.**

As Myriam I would like for members of the public to be able to donate to the library so that we can get extra funding to add more books.
