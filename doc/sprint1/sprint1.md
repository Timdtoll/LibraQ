**The user stories we decided to complete during this sprint are (total of 36 pts) :**   
As a user who has never been to the library I would like to create a Library account/card so that I can start borrowing books. (18 pts) 

- Implement user account creation feature where a user can sign up for a library account using their name, email address and password, and is given a unique identifier that is their library card number.

As John (a regular at the library) I would like to log in to the library website to check out books under my library account. (13 pts)

- Create log in feature where user can use their email and password to log in to their account

As John I would like to be able to search for books by any of the books attributes (Author/Title/Genre) so that I can find a book that matches my interests. (5 pts)

- Create search for books feature where a user can use a search bar and find books based on Author or Title or Genre (or any other book attribute)
