Sprint3 Planning Meeting

Participants: Taisia, Timothy, Akhanali, Parnia

The user stories we decided to complete during this sprint are (total of 43 pts) : 
As Myriam (the librarian) I would like to add or remove books from the collection displayed online to make damaged or missing books unavailable or make newly added books available. (6 pts)
As Myriam I would like to be able to mark a book as checked in online when someone brings it back in person, so that someone else can check it out later. (9 pts)
As John I would like to reserve an unavailable book so that I can read it as soon as it becomes available. (5 pts)
As John I would like to be able to extend a due date for a book I’ve checked out so that I have more time to read it in case I am not going to be done by the time it’s due. (10 pts)
As Myriam I want to be able to check which account has a particular book checked out at the moment so that I can get in contact with them in case they have long outstanding books or return a book damaged. (4 pts)
As a user who didn’t find the specific book they wanted I would like to request a book to be added to the library so that I can eventually read it without buying it. (9 pts)
